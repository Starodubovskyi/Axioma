# AXIOMA — IT Company Website

Сучасний корпоративний сайт IT-компанії AXIOMA, побудований на React + TypeScript + Tailwind CSS.

## Запуск

```bash
npm install
npm run dev
```

Відкрийте [http://localhost:5173](http://localhost:5173)

## Збірка

```bash
npm run build
npm run preview
```

## Секції

- **Hero** — головний екран з анімаціями та CTA
- **Stats** — ключові показники компанії
- **Services** — 9 послуг (сайти, e-commerce, боти, мікросервіси та ін.)
- **Projects** — портфоліо з модальними вікнами
- **Team** — команда експертів
- **Process** — етапи роботи
- **Contact** — форма зв'язку (десктоп + модалка на мобільному)

## Стек

- React 18 + TypeScript
- Vite 5
- Tailwind CSS 3
- Framer Motion
- Lucide Icons
